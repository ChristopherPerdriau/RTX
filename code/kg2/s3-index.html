Since KG2 includes UMLS and SNOMED CT, you need to contact the KG2 maintainer in order to demonstrate that you have accepted the license terms for these two knowledge-bases, in order to get access to download KG2. Once you have access, you would obtain them via S3 using the AWS CLI as follows:
  • KG2, before predicate-remapping and filtering: aws s3 cp --region us-west-2 s3://rtx-kg2/kg2.json.gz .
  • KG2, nodes only, before predicate-remapping and filtering: aws s3 cp --region us-west-2 s3://rtx-kg2/kg2-nodes.json.gz .
  • KG2, after predicate-remapping and filtering: aws s3 cp --region us-west-2 s3://rtx-kg2/kg2-simplified.json.gz .
