Documentation: Overview API Resources

The RTX team presents its pilot Reasoning Tool for evaluation. This page provides a simple web interface on top of the OpenAPI web service interface to RTX. At present only a few question types are supported. Select from one of the example queries to test the tool. Different drugs, diseases, etc. may be substituted for the default ones, but note that not all drugs, diseases, etc. are yet present in our knowledge graph.
